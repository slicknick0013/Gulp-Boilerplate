Typography

All typography and vertical rhythm uses base HTML elements (h1, h2, p, table, etc.) to set styles. The Sass module controlling this is a custom module found in /src/assets/sass/base/_typesetter.scss

Headings get a sans-serif font while main body text gets a serif treatment. Those fonts are managed as variables and can be found in /src/assets/sass/helpers/_variables.scss

Color

Horizonal Rule

The hr element inherits basic styling. It relies on the margins from the elements above and below it to determine it’s relationship in space.

The hr is a quick way to create separation inside content blocks.
